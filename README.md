# Terminal Snake
ASCII-based 'Snake Game' that runs inside a console like the Windows Terminal

## Getting Started


### Prerequisites
```
To run this program you need the *.NET Framework 4.5* (or higher)
```

### Note

```
**WORKS BEST IN WINDOWS TERMINAL**
```

### How To

```
Download the .NET Framework:

https://dotnet.microsoft.com/en-us/download/dotnet-framework

Then download SnakeNG.exe

```

### Controls 

```

*W,A,S,D* to move. That's it :)

```

### Troubleshooting

```

1. Resolution problems?

Optimal CMD properties:

Font-Size: 16
Font: Consolas
Buffer-Size: 120x9001
Windows-Size: 120x30
      
     
2. Characters not displaying correctly?

Run the game in ASCII mode:

SnakeNG.exe -a

```
