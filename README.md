# Terminal Snake

A snake game that runs purely on the terminal. Written in C#.  
Source is open.

![Alt Text](https://raw.githubusercontent.com/devdeuz/Terminal-Snake/main/img/demo.gif)

### Controls 

```

- [W][A][S][D] to move.

- [F5] to restart.

- [Esc] to exit.

- That's it :)

```

### Troubleshooting

```

Optimal settings:

Font-Size: 12
Font: Cascadia Mono
Windows-Size: 120x30

```
